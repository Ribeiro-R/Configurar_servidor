# Instalação Anaconda

Scripts para automatizar instalação do Anaconda.

## Instalação

<p>
Clone o repositório

```
git@github.com:Ribeiro-R/Scripts.git
```

Em seguida, vá para a pasta e torne o script executável.

```
cd Anaconda
$ chmod +x anaconda-install.sh
```

Agora é só executar.

```
./anaconda-install.sh
```

</p>
